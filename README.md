# kovski-portfolio-site

Portfolio site for Ben Velkovski — graphic designer specializing in brand identity systems, print production, and motion. Live at [kovskidesign.com](https://kovskidesign.com).

Static HTML/CSS/JS with no build step: open `index.html` in a browser, or serve the folder with any static server (e.g. `python3 -m http.server`).
